# 🧠 Neural Networks Lab

Repositório dedicado ao desenvolvimento, estudo e experimentação com **Redes Neurais Artificiais**, criado no contexto acadêmico para consolidar conceitos teóricos e aplicação prática em problemas reais.

---

## 📌 Objetivo

Este projeto tem como foco:

* Implementar redes neurais do zero (sem frameworks, quando possível)
* Utilizar bibliotecas como `TensorFlow`, `PyTorch` e `Scikit-learn`
* Explorar conceitos fundamentais como:

  * Perceptron
  * Redes Feedforward
  * Backpropagation
  * Funções de ativação
  * Overfitting vs Underfitting
* Aplicar modelos em datasets reais
